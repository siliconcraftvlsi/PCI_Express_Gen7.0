# PCI Express Gen7 Controller Known Issues

Generated: 2026-05-21

- This is a simulation-oriented controller model and is not a PCI-SIG compliance package.
- PIPE electrical/PHY behavior is represented by BFMs and digital models only.
- PCIe Gen7 timing, equalization, FLIT-mode, and certification-level details require additional work.
- Commercial simulator support is needed for the full UVM flow.

## Release Handling

Each known issue must be classified before release as fixed, accepted limitation, deferred feature, or waiver. Accepted limitations must also appear in the project report or release notes.
